# TEST QUALITY ASSURANCE ENGINEER

## website : https://demo.midtrans.com/

### Implementasi :
- Mengimplementasikan browser Chrome dalam mode incognito
- Data test menggunakan DDT
- Mengimplementasikan WebDriverWait
- Mengimplementasikan time.sleep untuk melihat inputan/untuk hold halaman sementara untuk melihat hasilnya, time.sleep bisa di skip karena sudah mengimplementasikan WebDriverWait
- Mengimplementasikan Assertion

### Cara Running :
- Jika menggnukan Virtual Environment, aktifkan terlebih dahulu melalui terminal CMD/PowerShell.
- CMD         : .venv\Scripts\activate.bat
- PowerShell  : .venv\Scripts\Activate.ps1
- Untuk running program sesuaikan dengan folder dan filenya, contoh : pytest -v -s namafolder\namafile.py
